About This Demo

This demo displays real-time sensor data from your device. Note: Desktop computers typically don't have motion sensors - you'll need a smartphone or tablet to see orientation, acceleration, and gyroscope data.

iOS Setup

For GPS location on iPhone:

  1. Go to Settings → Safari → Location
  2. Set to "Ask" or "Allow"
  3. Refresh this page and click "Enable GPS"
  4. Tap "Allow" when Safari prompts for location access

Sensor Types

  • Orientation: Device rotation in 3D space (mobile only)
  • Acceleration: Linear movement of the device (mobile only)
  • Gyroscope: Rotational velocity (mobile only)
  • GPS Velocity: Speed and heading from GPS (requires movement)
  • GPS Position: Latitude, longitude, and accuracy
