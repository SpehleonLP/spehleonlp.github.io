About This Demo

This demo displays real-time sensor data from your device. On iOS, you may need to use the Mozilla WebXR Viewer for full sensor access.

  • Orientation: Device rotation in 3D space
  • Acceleration: Linear movement of the device
  • Gyroscope: Rotational velocity
  • GPS Velocity: Speed and heading from GPS (Doppler effect)
  • GPS Position: Latitude, longitude, and accuracy
