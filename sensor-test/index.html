About This Demo

This demo uses WebXR Device API to access device sensors. On iOS, use the Mozilla WebXR Viewer for full compatibility.

  • AR Mode: Uses device camera and sensors for augmented reality
  • VR Mode: Immersive virtual reality experience
  • Sensors: Orientation, acceleration, gyroscope, and position tracking
